# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Architecture diagram parser with Mermaid-style syntax support
- BFS-based constraint layout algorithm for auto-positioning
- Lottie animation renderer with ByteByteGo-inspired styling
- Layered architecture diagrams with horizontal flow (input → process → action → output)
- 25+ icons support (brain, gear, database, server, api, cloud, etc.)
- Mindmap parser and renderer
- Electron GUI with interactive mindmap editor
- CLI for generating Lottie animations from .kore files
- Sample diagrams (agentic_ai.kore, mindmap samples)
- Preview HTML files for viewing animations
- Test suite for mindmap CLI commands
