# Mindmap

Mindmap là phương pháp ghi chú trực quan với ý tưởng trung tâm ở giữa và các nhánh tỏa ra xung quanh. Tony Buzan phổ biến kỹ thuật này từ những năm 1970, nhấn mạnh việc sử dụng màu sắc, hình ảnh và từ khóa để kích thích tư duy. Cấu trúc phân cấp của mindmap phản ánh cách não bộ tổ chức thông tin theo associations.

## Commercial

**XMind** - Phần mềm mindmap phổ biến nhất thế giới với hàng triệu người dùng. Nổi bật với Zen Mode giúp tập trung vào nội dung, loại bỏ distractions - triết lý tương đồng với Kore về sự tối giản. Hỗ trợ nhiều structure types (logic chart, org chart, tree, fishbone) và export sang SVG/PNG với chất lượng cao.

**MindMeister** - Tiên phong trong web-based mindmap với 27+ triệu users. Điểm mạnh là presentation mode biến mindmap thành slideshow tự động - gợi ý cho Kore về animation sequence. Markdown syntax trong nodes cho phép formatting nhanh mà không rời tay khỏi keyboard.

**MindManager** - Chuẩn enterprise với tích hợp sâu vào Microsoft 365 và project management. Tính năng SmartRules tự động format nodes theo conditions - concept có thể áp dụng cho Kore về conditional styling. Export HTML5 interactive cho phép share mindmap động không cần cài đặt.

## Open Source

**FreeMind** - Phần mềm mindmap open source đầu tiên (2003), nền tảng cho nhiều tools khác. Triết lý "one key, one action" với keyboard shortcuts tối ưu - ảnh hưởng đến thiết kế hotkeys của Kore. File format XML đơn giản, dễ parse và transform programmatically.

**Freeplane** - Fork từ FreeMind với scripting engine mạnh mẽ bằng Groovy. Hỗ trợ formulas trong nodes, biến mindmap thành công cụ tính toán - mở rộng khái niệm "thinking tool". Add-on system cho phép community đóng góp features mà không thay đổi core.

**SimpleMindMap** - Modern web-based editor với 10k+ GitHub stars, viết bằng JavaScript. Source code clean, well-documented - tham khảo tốt cho Kore về SVG rendering và layout algorithms. Hỗ trợ custom themes qua JSON config và export nhiều định dạng vector.
